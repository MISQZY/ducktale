## `/towny` — Общая информация (все игроки)
 
| Команда | Задача | Роль |
|---|---|---|
| `/towny map` | Показать карту мира с отмеченными городами | Все |
| `/towny hud` | Включить карту в боковой панели (scoreboard) | Все |
| `/towny prices` | Показать налоги и стоимость содержания города | Все |
| `/towny time` | Показать время до следующего дня (сбор налогов) | Все |
| `/towny universe` | Показать глобальную статистику: города, нации, жители | Все |
| `/towny top residents {all/town/nation}` | Рейтинг по количеству жителей | Все |
| `/towny top land {all/resident/town}` | Рейтинг по количеству земли | Все |
| `/towny spy` | Следить за всеми каналами чата | Администратор |

## `/plot` — Управление участками
 
| Команда | Задача | Роль |
|---|---|---|
| `/plot claim` | Лично купить участок, выставленный на продажу | Житель |
| `/plot claim auto` | Автоматически купить несколько участков вокруг | Житель |
| `/plot unclaim` | Отказаться от личного участка | Житель |
| `/plot forsale $` | Выставить участок на продажу по цене | Владелец участка / Мэр |
| `/plot notforsale` | Убрать участок с продажи | Владелец участка / Мэр |
| `/plot evict` | Выселить жителя с его личного участка | Мэр / Помощник |
| `/plot evict forsale` | Выселить жителя и сразу выставить участок на продажу | Мэр / Помощник |
| `/plot set shop` | Превратить участок в магазин | Мэр |
| `/plot set embassy` | Превратить участок в посольство | Мэр |
| `/plot set arena` | Превратить участок в арену (PvP) | Мэр |
| `/plot set farm` | Превратить участок в ферму | Мэр |
| `/plot set jail` | Превратить участок в тюрьму | Мэр |
| `/plot set wilds` | Превратить участок в дикую зону | Мэр |
| `/plot set inn` | Превратить участок в постоялый двор | Мэр |
| `/plot set bank` | Превратить участок в банк | Мэр |
| `/plot set outpost` | Превратить участок в аванпост | Мэр |
| `/plot set reset` | Сбросить тип участка обратно в обычный | Мэр |
| `/plot set name {название}` | Дать участку своё название | Мэр / Владелец |
| `/plot set perm ...` | Настроить права доступа на участке | Владелец участка / Мэр |
| `/plot toggle fire` | Включить / выключить распространение огня на участке | Владелец |
| `/plot toggle pvp` | Включить / выключить PvP на участке | Владелец |
| `/plot toggle explosion` | Включить / выключить взрывы на участке | Владелец |
| `/plot toggle mob` | Включить / выключить спавн мобов на участке | Владелец |
| `/plot trust add {игрок}` | Добавить игрока как доверенного на участке | Владелец |
| `/plot trust remove {игрок}` | Убрать игрока из доверенных на участке | Владелец |
| `/plot perm gui` | Открыть GUI для настройки прав участка | Владелец |
| `/plot group add {название}` | Создать группу участков / добавить в существующую | Мэр / Владелец |
| `/plot group forsale {цена}` | Выставить группу участков на продажу | Мэр / Владелец |
| `/plot district create {название}` | Создать район из участков | Мэр |
| `/plot district add {название}` | Добавить участок в район | Мэр |
| `/plot jailcell add` | Добавить камеру в тюремный участок | Мэр |
| `/plot clear` | Очистить блоки на участке по заданному списку | Мэр / Владелец |
