# 🛡️ Команды кланов

## Основные

| Команда | Описание |
|---------|----------|
| `/clan create [название]` | Создать клан (1 000 🪙) |
| `/clan disband` | Распустить клан |
| `/clan info` | Информация о своём клане |
| `/clan info [название]` | Информация о другом клане |
| `/clan list` | Список всех кланов |

## Участники

| Команда | Описание |
|---------|----------|
| `/clan invite [ник]` | Пригласить игрока |
| `/clan join [название]` | Принять приглашение |
| `/clan leave` | Покинуть клан |
| `/clan kick [ник]` | Исключить игрока |
| `/clan promote [ник]` | Повысить до офицера |
| `/clan demote [ник]` | Понизить до рядового |

## Территории

| Команда | Описание |
|---------|----------|
| `/clan claim` | Застолбить чанк |
| `/clan unclaim` | Освободить чанк |
| `/clan map` | Карта клановых земель |

## Казна

| Команда | Описание |
|---------|----------|
| `/clan deposit [сумма]` | Пополнить казну |
| `/clan withdraw [сумма]` | Снять из казны (офицер+) |
| `/clan bank` | Баланс казны |
