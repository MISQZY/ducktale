# 💰 Команды экономики

## Магазин

| Команда | Описание |
|---------|----------|
| `/shop` | Открыть магазин |
| `/sell hand` | Продать предмет в руке |
| `/sell all` | Продать весь инвентарь |
| `/buy [предмет] [кол-во]` | Купить предмет |

## Аукцион

| Команда | Описание |
|---------|----------|
| `/ah` | Открыть аукцион |
| `/ah sell [цена]` | Выставить предмет |
| `/ah cancel` | Снять свой лот |
| `/ah search [запрос]` | Поиск по аукциону |

## Банк

| Команда | Описание |
|---------|----------|
| `/bank` | Открыть меню банка |
| `/bank deposit [сумма]` | Положить монеты |
| `/bank withdraw [сумма]` | Снять монеты |
| `/bank balance` | Баланс счёта |

## Переводы

| Команда | Описание |
|---------|----------|
| `/pay [ник] [сумма]` | Перевести монеты игроку |
| `/balance` | Узнать свой баланс |
| `/baltop` | Топ богатейших игроков |
