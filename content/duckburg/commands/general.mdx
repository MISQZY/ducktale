# 📋 Основные команды

## Телепортация

| Команда | Описание |
|---------|----------|
| `/spawn` | Вернуться на спавн |
| `/home` | Телепорт домой |
| `/sethome` | Установить точку дома |
| `/delhome` | Удалить точку дома |
| `/tpa [ник]` | Запрос телепорта к игроку |
| `/tpaccept` | Принять запрос телепорта |
| `/warp [название]` | Телепорт к варпу |
| `/warps` | Список всех варпов |

## Общение

| Команда | Описание |
|---------|----------|
| `/msg [ник] [текст]` | Личное сообщение |
| `/r [текст]` | Ответить на ЛС |
| `/ignore [ник]` | Игнорировать игрока |

## Прочее

| Команда | Описание |
|---------|----------|
| `/balance` | Узнать баланс |
| `/pay [ник] [сумма]` | Перевести монеты |
| `/stats` | Статистика твоего персонажа |
| `/help` | Справка |
| `/rules` | Правила сервера |
