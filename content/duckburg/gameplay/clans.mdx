# 🛡️ Кланы

Кланы — объединения игроков с общими землями, казной и целями.

## Создание клана

Стоимость создания: **1 000 🪙**

```
/clan create [название]
```

Название клана: от 3 до 16 символов, только буквы и цифры.

## Управление

### Роли

| Роль | Права |
|------|-------|
| Лидер | Все права |
| Офицер | Приглашать, кикать рядовых, управлять землёй |
| Рядовой | Строить на клановой земле, использовать казну |

### Команды лидера

```
/clan invite [ник]       — пригласить игрока
/clan kick [ник]         — исключить
/clan promote [ник]      — повысить до офицера
/clan disband            — расформировать клан
```

## Территории

Захватывай землю командой `/clan claim` стоя на нужном чанке. Стоимость чанка — **50 🪙/неделю**.

### Защита земли

На клановой земле чужие игроки **не могут**:
- Ломать и ставить блоки
- Открывать сундуки и двери
- Убивать животных

<Callout type="warning">
  Если у клана закончатся монеты в казне, земли станут незащищёнными. Пополняй казну командой `/clan deposit`.
</Callout>
