# 💰 Экономика

DuckBurg использует кастомную экономику на основе **Duck Coins (🪙)**.

## Получение монет

| Способ | Доход |
|--------|-------|
| Продажа ресурсов в `/shop` | Низкий, стабильный |
| Аукцион `/ah` | Высокий, зависит от рынка |
| Квесты | Средний + уникальные предметы |
| PvP турниры | Высокий, нестабильный |
| Работа в кланах | Зависит от клана |

## Магазин `/shop`

Базовый магазин с фиксированными ценами. Доступен всегда:

```
/shop           — открыть магазин
/sell hand      — продать предмет в руке
/sell all       — продать всё в инвентаре
```

## Аукцион `/ah`

Игровой рынок. Ты сам устанавливаешь цену:

```
/ah             — открыть аукцион
/ah sell [цена] — выставить предмет в руке
/ah cancel      — снять свой лот
```

## Банк `/bank`

Храни монеты безопасно:

```
/bank deposit [сумма]   — положить
/bank withdraw [сумма]  — снять
/bank balance           — баланс
```

<Callout type="tip">
  Следи за курсом редких ресурсов — цены на аукционе меняются в зависимости от сезона!
</Callout>
