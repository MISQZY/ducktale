## Требования

- Лицензионный Minecraft Java Edition
- Версия игры: **1.21.10**

## Шаги подключения

### 1. Открой Minecraft

Запусти лаунчер и убедись, что выбрана версия **1.21.10**.

### 2. Перейди в «Мультиплеер»

На главном экране нажми **«Мультиплеер»** → **«Добавить сервер»**.

### 3. Введи адрес

| Поле | Значение |
|------|----------|
| Название | DuckTale — DuckBurg |
| Адрес | `s6.yufu.su:25569` |

### 4. Подключись

Нажми **«Готово»**, затем дважды кликни на сервер в списке.

## Вход в учетную запись сервера
<Callout type="info">
  Если ты играешь с лицензионной версии игры - можешь смело пропускать этот шаг!
</Callout>

При первом входе если ты играешь с **пиратской версии** игры - попросит зарегистрироваться на сервере. Для регистрации используй команду:
```
/register <Пароль> <Повтори пароль>
```
<Callout type="warn">
  Не забудь этот пароль. Он потребуется при каждом входе на сервер!
</Callout>
При повторных входах используй для авторизации команду:
```
/login <Пароль>
```
