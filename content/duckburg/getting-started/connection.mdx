# 🔗 Подключение к серверу

## Требования

- Лицензионный Minecraft Java Edition
- Версия клиента: **1.21.x** (рекомендуется последняя)

## Шаги подключения

### 1. Открой Minecraft

Запусти лаунчер и убедись, что выбрана версия **1.21.x**.

### 2. Перейди в «Мультиплеер»

На главном экране нажми **«Мультиплеер»** → **«Добавить сервер»**.

### 3. Введи адрес

| Поле | Значение |
|------|----------|
| Название | DuckTale — DuckBurg |
| Адрес | `s6.yufu.su:25569` |

### 4. Подключись

Нажми **«Готово»**, затем дважды кликни на сервер в списке.

## Первый вход

При первом входе тебя встретит обучающий квест. Пройди его — получишь стартовый набор предметов и **500 монет** на первые покупки.

<Callout type="info">
  Если сервер недоступен — проверь статус в нашем [Discord](https://discord.gg/ducktale).
</Callout>
