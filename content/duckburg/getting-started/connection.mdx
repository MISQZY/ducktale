## Требования

- Minecraft **Java Edition**
- Версия игры: **1.21.10**

<Callout variant="info">
  Более старые версии могут не поддерживаться. Если не уверен — обновись до актуальной.
</Callout>

## Авторизация

<Callout variant="tip" title="Лицензия">
  Если ты играешь с лицензионной версии Minecraft — этот раздел можно пропустить. Авторизация происходит автоматически.
</Callout>

При первом входе с **пиратской версии** игры сервер попросит зарегистрироваться:

```
/register <Пароль> <Повтори пароль>
```

<Callout variant="warning">
  Запомни пароль — он потребуется при каждом входе на сервер.
</Callout>

При повторных входах используй:

```
/login <Пароль>
```
