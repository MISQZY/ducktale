# Получение проходки

<Callout type="warn">
  Получение проходки на сервер является важнейшим этапом для игры.
</Callout>

Для прохода на сервер тебе важно получить проходку на твой ник. Сделать это можно следующими способами:
- Попросить добавить тебя у любого медийного лица сервера (например [6oJIeH](https://www.twitch.tv/6ojieh)).
- Попросить уже играющего друга добавить тебя на сервер. Выполнение команды стоит *500* Дукатов. Другу необходимо использовать команду `/invitefriend <Твой Ник>`.
- Попросить добавить тебя у администрациии сервера (Модератора, Аадминистратор)
