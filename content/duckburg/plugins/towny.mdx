## Как устроены города

Основная единица Towny — **чанк** (16×16 блоков). Мэр «застолбил» чанки — они защищены: чужие игроки не могут ломать блоки без разрешения.

Каждый чанк стоит золото из городской казны. Если казна опустеет — город распускается автоматически. Следи за балансом.

### Типы участков

| Тип | Назначение |
|-----|-----------|
| **Обычный** | Жилой участок, выдаётся жителям |
| **Магазин** | Торговая зона, разрешена установка сундуков-магазинов |
| **Ферма** | Сельскохозяйственная зона |
| **Арена** | PvP разрешён по умолчанию |
| **Тюрьма** | Для заключения нарушителей |
| **Посольство** | Нейтральная зона для игроков других городов |
| **Аванпост** | Удалённый форпост города, не обязан граничить с основной территорией |

## Нации

Несколько городов могут объединиться в **нацию**. Столичный мэр становится королём. Нации могут:

- Заключать союзы (`/nation ally add <нация>`)
- Объявлять врагов (`/nation enemy add <нация>`)
- Устанавливать налоги для входящих городов
- Объявлять нейтралитет (за дополнительную плату)

<Callout variant="info">
  Все команды Towny — в разделе [Команды → Towny](/docs/duckburg/commands/towny).
</Callout>
